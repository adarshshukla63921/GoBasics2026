# Basic Types In Go-lang

These are the following basic types in Go.

1) Booleans : `bool`  
2) Strings : string  
3) Integers : int8, int16, int32, int64  
4) Decimal Numbers : float32, float64  
5) Complex Numbers : complex64, complex 128  

byte alias for `uint8`  
rune alias for `int32`  

## Zero Values for basic type

| Type | Zero Value |
|------|------------|
| `bool` | `false` |
| `string` | `""` |
| `int` | `0` |
| `int8` | `0` |
| `int16` | `0` |
| `int32` | `0` |
| `int64` | `0` |
| `uint` | `0` |
| `uint8` (`byte`) | `0` |
| `uint16` | `0` |
| `uint32` | `0` |
| `uint64` | `0` |
| `uintptr` | `0` |
| `float32` | `0.0` |
| `float64` | `0.0` |
| `complex64` | `0 + 0i` |
| `complex128` | `0 + 0i` |
| `rune` (`int32`) | `0` |
